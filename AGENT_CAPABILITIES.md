# Agent Capabilities & Task Examples

## Overview
Each agent in the naiborhood platform can execute autonomous tasks, generate real outputs (files, reports, documents), and provide tangible results - not just text responses.

## How It Works

### Autonomous Mode
When autonomous mode is enabled (ON by default), agents can:
1. **Detect task intent** from your natural language requests
2. **Execute multi-step processes** independently
3. **Generate downloadable files** (PDF, Excel, Word, CSV)
4. **Show progress updates** in real-time
5. **Notify you when complete** with actionable outputs

## Financial Analyst Agent

### What It Can Do
- Generate financial reports (daily, weekly, monthly)
- Analyze revenue trends and patterns
- Categorize expenses automatically
- Detect financial anomalies
- Create cash flow forecasts
- Produce downloadable Excel/PDF reports

### Task Examples

#### Generate Reports
```
"Generate a monthly financial report for December"
"Create a Q4 revenue analysis"
"Give me a daily sales summary"
```

**Output**: Downloadable Excel file with metrics, charts, and insights

#### Expense Analysis
```
"Categorize last month's expenses"
"Show me spending trends for Q4"
"Which categories are over budget?"
```

**Output**: Categorized expense breakdown with recommendations

#### Anomaly Detection
```
"Check for unusual transactions this month"
"Are there any financial red flags I should know about?"
```

**Output**: List of anomalies with severity ratings and explanations

#### Forecasting
```
"Forecast cash flow for next 3 months"
"Predict Q1 revenue based on current trends"
```

**Output**: Forecasting report with predictions and confidence levels

### What It Can Do
- Extract vendor/supplier information from documents
- Summarize contracts and agreements
- Extract key clauses
- Compare document versions
- Assess risks and compliance issues
- Search for specific terms

### Task Examples

#### Vendor Extraction
```
"Extract all vendor information from this contract"
```
