# protocolLM

## Overview
protocolLM is a Next.js application that helps food service teams identify potential health code issues before inspections. The app lets users upload photos or ask questions and receives structured guidance based on the Michigan Food Code and local Washtenaw County requirements.

## Installation
1. Ensure you have Node.js 20+ and npm 10+ installed.
2. Install dependencies:
   ```bash
   npm install
   ```

## Running Locally
Start the development server:
```bash
npm run dev
```
Then open the app in your browser at http://localhost:3000.

## What is protocolLM?
protocolLM analyzes images and questions against the Michigan Food Code and Washtenaw guidance to surface likely violations and corrective steps. It is a reference tool designed to help establishments prepare for health inspections.
