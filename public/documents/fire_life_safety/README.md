# Fire & Life Safety - Occupied Buildings

## Sector Overview
This sector covers fire code compliance and life safety requirements for occupied buildings in Michigan.

## Planned Documents
Documents to be ingested for this sector include:
- Michigan Building Code (Fire Safety provisions)
- NFPA 1: Fire Code
- NFPA 101: Life Safety Code
- Michigan Fire Prevention Code
- Exit requirements and egress standards
- Fire suppression system requirements
- Emergency lighting and signage
- Fire alarm system requirements
- Occupancy classifications and limits

## Ingestion
Once documents are added to this folder, run:
```bash
npm run ingest -- --sector=fire_life_safety
```

## Document Format
- PDF format preferred
- Named descriptively (e.g., `Michigan_Fire_Prevention_Code.pdf`)
- Organized by regulation type or code section
