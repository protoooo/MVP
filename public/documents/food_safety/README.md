# Food Safety Sector

## Sector Overview
Food service establishments, restaurants, and health department inspections in Michigan.

## Status
âœ… **Active** - This sector is fully operational.

## Documents
This folder contains all Michigan food safety regulatory documents including:
- Michigan Modified Food Code
- Temperature control guidelines
- Food employee illness guidelines
- Consumer advisories
- Emergency action plans
- Date marking guidelines

## Ingestion
To re-ingest these documents:
```bash
npm run ingest -- --sector=food_safety
```

## Legacy Compatibility
This sector previously used the `michigan` folder name and county-based scoping (washtenaw, wayne, oakland, macomb). The system maintains backward compatibility with these legacy identifiers.
